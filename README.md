# Моделирование и анализ надежности системы с резервированием на основе непрерывных марковских цепей
Программа предназначена для моделирования системы из двух типов устройств с переменным количеством резервов,
задаваемыми параметрами интенсивности отказов и интенсивности ремонта (при одном ремонтнике).
По заданным параметрам системы программа строит граф состояний системы, реализует три способа нахождения вероятностей состояний системы:
- С помощью численного решения системы дифференциальных уравнений Колмогорова;
- С помощью имитационного моделирования в терминах непрерывных марковских цепей;
- С помощью имитационного моделирования в терминах дискретно-событийного моделирования.4
